## E-INK OPTIMIZATION NOTES
## ────────────────────────

# Design Principles:
# - Use high contrast (pure black on white)
# - Avoid fine details that may blur
# - Keep line width ≥ 2 pixels equivalent
# - Minimize partial refresh areas
# - Group related elements for single refresh
# - Use solid fills sparingly (battery drain)

# Character Selection:
# - Box drawing: ─ │ ┌ ┐ └ ┘ ├ ┤ ┬ ┴ ┼
# - Double box: ═ ║ ╔ ╗ ╚ ╝ ╠ ╣ ╦ ╩ ╬
# - Blocks: █ ▄ ▀ ░ ▒ ▓
# - Arrows: ← → ↑ ↓ ◄ ► ▲ ▼
# - Symbols: ◆ ◇ ○ ● □ ■ ☞ ▸
# - Medieval: ⚔ 🛡 📜 ♔ ✧ ⚡
